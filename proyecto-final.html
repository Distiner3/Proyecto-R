Proyecto Final: Análisis de Datos Completo

¡Llegó el momento de aplicar todo lo que has aprendido! En este proyecto integrador realizarás un análisis completo de datos desde la carga hasta el reporte final.

Proyecto Capstone 2-4 horas Individual

Objetivo del Proyecto

Realizarás un análisis estadístico completo de un dataset real, aplicando todas las técnicas aprendidas en el curso. Tu entregable final será un reporte profesional con código, visualizaciones y conclusiones.

¿Qué vas a lograr?
  • Demostrar dominio de R para análisis de datos
  • Aplicar conceptos estadísticos en un caso real
  • Crear visualizaciones profesionales
  • Redactar conclusiones basadas en evidencia
  • Desarrollar un portafolio de trabajo
1
Fase 1: Selección del Dataset

Elige uno de estos datasets reales para realizar tu análisis:

Intermedio
Análisis de Ventas

Dataset con 2,500 registros de ventas mensuales por producto, región y vendedor. Incluye variables categóricas y numéricas.

Variables: Fecha, Producto, Vendedor, Región, Ventas, Comisión
Avanzado
Indicadores de Salud

Datos de 1,800 pacientes con indicadores vitales, hábitos y diagnósticos. Ideal para correlaciones complejas.

Variables: Edad, IMC, Presión, Ejercicio, Tabaquismo, Diagnóstico
Intermedio
Desempeño Académico

Calificaciones de 1,200 estudiantes con factores socioeconómicos y de estudio. Perfecto para principiantes.

Variables: Calificaciones, Horas de estudio, Ingreso familiar, Apoyo parental
Experto
Mercado Inmobiliario

Precios de 3,000 propiedades con características detalladas. Incluye análisis de regresión y predicción.

Variables: Precio, m², Habitaciones, Ubicación, Antigüedad, Amenidades

Fase 3: Análisis Estadístico Profundo

Aplica las técnicas estadísticas aprendidas para extraer insights valiosos:

Análisis Requeridos:
Ejemplo de Análisis:
# Análisis por grupos
datos %>%
  group_by(categoria) %>%
  summarise(
    media = mean(variable),
    mediana = median(variable),
    sd = sd(variable)
  )

# Matriz de correlaciones
cor(datos[, c("var1", "var2", "var3")])
Objetivos de esta Fase

Al completar esta fase, deberías poder responder preguntas específicas sobre tu dataset y tener insights claros que puedas comunicar a otros.

4
Fase 4: Visualizaciones Profesionales

Crea gráficos que comuniquen tus hallazgos de manera clara y atractiva:

Gráficos Requeridos:
Criterios de Calidad:
  • Títulos descriptivos: Cada gráfico debe tener un título claro
  • Ejes etiquetados: Con unidades cuando sea apropiado
  • Colores apropiados: Que faciliten la interpretación
  • Leyendas claras: Cuando uses múltiples series
  • Escalas apropiadas: Que no distorsionen la información
# Ejemplo de gráfico profesional
ggplot(datos, aes(x = variable_x, y = variable_y)) +
  geom_point(aes(color = categoria), size = 3) +
  labs(
    title = "Título Descriptivo",
    x = "Etiqueta del Eje X",
    y = "Etiqueta del Eje Y"
  ) +
  theme_minimal()
