Fundamentos de Estadística Descriptiva con R

La estadística descriptiva nos ayuda a resumir y entender nuestros datos. En esta sección aprenderás los conceptos más importantes y cómo aplicarlos en R con ejemplos prácticos que puedes seguir paso a paso.

Objetivo: Al final de esta sección, sabrás calcular e interpretar las medidas estadísticas más importantes para describir cualquier conjunto de datos.
Medidas de Tendencia Central

Las medidas de tendencia central nos indican el "centro" o valor típico de nuestros datos. Son las estadísticas más básicas e importantes.

# Datos de ejemplo: calificaciones de 10 estudiantes
calificaciones <- c(85, 90, 78, 92, 88, 76, 94, 87, 83, 91)
calificaciones
Media Aritmética
86.4
x̄ = Σx / n
mean(calificaciones)
[1] 86.4
Mediana
87.5
Valor central ordenado
median(calificaciones)
[1] 87.5
Moda
N/A
Valor más frecuente
# No hay moda en este caso
# (todos los valores aparecen una vez)
¿Cuándo usar cada una?
  • Media: Cuando los datos son simétricos y no hay valores extremos
  • Mediana: Cuando hay valores extremos o la distribución es asimétrica
  • Moda: Para datos categóricos o cuando quieres el valor más común
Ejemplo Práctico

Situación: Si tienes salarios de [2000, 2100, 2050, 2200, 15000], la media sería 4,670 (no representativa), pero la mediana sería 2,100 (más representativa del salario típico).

Medidas de Dispersión

Las medidas de dispersión nos dicen qué tan "esparcidos" están los datos. ¿Los valores están concentrados cerca del centro o muy dispersos?

Rango
18
R = Máximo - Mínimo
max(calificaciones) - min(calificaciones)
[1] 18 # (94 - 76)
Varianza
35.6
σ² = Σ(x - x̄)² / (n-1)
var(calificaciones)
[1] 35.6
Desviación Estándar
5.97
σ = √σ²
sd(calificaciones)
[1] 5.97
Coeficiente de Variación
6.9%
CV = (σ / x̄) × 100
(sd(calificaciones) / mean(calificaciones)) * 100
[1] 6.9
Interpretación
  • Desviación Estándar = 5.97: En promedio, las calificaciones se desvían 5.97 puntos de la media
  • CV = 6.9%: Baja variabilidad (< 15% se considera baja)
  • Regla 68-95-99.7: ~68% de datos están dentro de ±1σ de la media
Cuartiles y Percentiles

Los cuartiles dividen los datos en 4 partes iguales. Son especialmente útiles para entender la distribución y detectar valores atípicos.

# Calcular cuartiles
cuartiles <- quantile(calificaciones)
cuartiles

# 0% 25% 50% 75% 100%
# 76 81.5 87.5 91.0 94

Detección de Valores Atípicos:
# Límites para outliers
Q1 <- quantile(calificaciones, 0.25)
Q3 <- quantile(calificaciones, 0.75)
IQR_val <- Q3 - Q1

limite_inferior <- Q1 - 1.5 * IQR_val
limite_superior <- Q3 + 1.5 * IQR_val

# Outliers: valores fuera de estos límites
Interpretación Práctica
  • 25% de estudiantes sacaron 81.5 o menos
  • 50% de estudiantes sacaron entre 81.5 y 91.0
  • 75% de estudiantes sacaron 91.0 o menos
  • IQR = 9.5: El 50% central tiene un rango de 9.5 puntos
Correlación y Covarianza

La correlación mide qué tan relacionadas están dos variables. Es fundamental para entender relaciones entre diferentes características de nuestros datos.

# Datos de ejemplo: horas de estudio vs calificaciones
horas_estudio <- c(2, 4, 3, 6, 5, 1, 7, 4, 3, 6)
calificaciones <- c(78, 85, 80, 92, 88, 70, 94, 83, 79, 90)

# Correlación de Pearson
correlacion <- cor(horas_estudio, calificaciones)
correlacion
[1] 0.897 # Correlación fuerte positiva
Interpretación de la Correlación:
Valor r Interpretación Ejemplo Visual
r ≈ +1 Correlación positiva perfecta ↗
+0.7 a +0.9 Correlación positiva fuerte ↗
+0.3 a +0.7 Correlación positiva moderada ↗
r ≈ 0 Sin correlación ↔
r ≈ -1 Correlación negativa perfecta ↘
Covarianza:
# Covarianza
cov(horas_estudio, calificaciones)
[1] 21.67

# Interpretación:
# Positiva = varían juntas
# Negativa = varían opuestas
r = Cov(X,Y) / (σₓ × σᵧ)
En nuestro ejemplo

r = 0.897 indica una correlación muy fuerte entre horas de estudio y calificaciones. Esto sugiere que los estudiantes que estudian más tienden a obtener mejores calificaciones.

¡Importante: Correlación NO implica causalidad. Solo indica que las variables cambian juntas.
