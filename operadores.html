¿Qué son los Operadores?

Los operadores son símbolos especiales que realizan operaciones específicas en R. Son la base para hacer cálculos, comparaciones y operaciones lógicas.

Consejo: Prueba cada ejemplo en tu consola de RStudio para ver los resultados en tiempo real.
Operadores Aritméticos

Realizan operaciones matemáticas básicas

Operador Descripción Ejemplo Resultado
+ Suma 5 + 3 8
- Resta 10 - 4 6
* Multiplicación 6 * 7 42
/ División 15 / 3 5
^ Potencia 2 ^ 3 8
%% Módulo (resto) 17 %% 5 2
%/% División entera 17 %/% 5 3
Ejemplo Práctico

Vamos a calcular el área de un círculo con radio 5:

# Definir el radio
radio <- 5
# Calcular el área (π * r²)
area <- 3.14159 * (radio ^ 2)
area
[1] 78.53975
Operadores de Comparación

Comparan valores y devuelven TRUE o FALSE

Operador Descripción Ejemplo Resultado
> Mayor que 8 > 5 TRUE
< Menor que 3 < 7 TRUE
>= Mayor o igual que 10 >= 10 TRUE
<= Menor o igual que 4 <= 3 FALSE
== Igual a 5 == 5 TRUE
!= Diferente de 9 != 6 TRUE
Ejemplo Práctico

Comparemos edades para determinar si alguien es mayor de edad:

# Definir edades
edad_ana <- 17
edad_carlos <- 22
# Verificar si son mayores de edad
ana_mayor_edad <- edad_ana >= 18
carlos_mayor_edad <- edad_carlos >= 18
# Ver resultados
ana_mayor_edad
carlos_mayor_edad
[1] FALSE
[1] TRUE
Operadores Lógicos

Combinan condiciones lógicas (TRUE/FALSE)

Operador Descripción Ejemplo Resultado
! NO lógico (negación) !TRUE FALSE
& Y lógico (elemento por elemento) TRUE & FALSE FALSE
| O lógico (elemento por elemento) TRUE | FALSE TRUE
&& Y lógico (solo primer elemento) TRUE && FALSE FALSE
|| O lógico (solo primer elemento) TRUE || FALSE TRUE
Importante: Para usar operadores lógicos con múltiples condiciones, siempre usa paréntesis: (edad > 18 & sueldo > 1000)
Ejemplo Práctico

Determinemos si una persona califica para un préstamo:

# Definir datos del solicitante
edad <- 25
ingresos <- 2500
tiene_trabajo <- TRUE
# Criterios: mayor de 21 Y ingresos > 2000 Y tiene trabajo
califica <- (edad > 21) & (ingresos > 2000) & tiene_trabajo
califica

# También podemos verificar si NO califica
no_califica <- !califica
no_califica
[1] TRUE
[1] FALSE

Ejercicio 1: Operaciones Básicas
# Calcula:
a <- 15
b <- 4
# a) La suma de a y b
# b) a elevado a la potencia b
# c) El resto de dividir a entre b
Ejercicio 2: Condiciones
# Una tienda da descuento si:
# - Compra > $100 Y es cliente VIP
# - O si compra > $500
compra <- 350
es_vip <- FALSE
# ¿Obtiene descuento?
Soluciones:
suma <- a + b # 19
potencia <- a ^ b # 50625
resto <- a %% b # 3
descuento <- ((compra > 100) & es_vip) | (compra > 500)
descuento # FALSE
