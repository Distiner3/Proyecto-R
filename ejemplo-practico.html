Análisis Estadístico Completo: Estaturas de Estudiantes

En este ejemplo práctico vamos a realizar un análisis estadístico completo de un conjunto de datos sobre estaturas de estudiantes. Aprenderás a calcular medidas de tendencia central, dispersión, y crear visualizaciones profesionales.

Objetivo: Analizar las estaturas de 150 estudiantes universitarios para entender la distribución de alturas en nuestra muestra.
1
Preparación: Instalación de Librerías

Primero instalamos las librerías necesarias para nuestro análisis:

# Instalar librerías necesarias
install.packages("DT")        # Para tablas interactivas
install.packages("dplyr")     # Para manipulación de datos
install.packages("DescTools") # Para estadísticas descriptivas
install.packages("ggplot2")   # Para gráficos avanzados

# Cargar las librerías
library(DT)
library(dplyr)
library(DescTools)
library(ggplot2)
¿Por qué estas librerías?
  • DT: Crea tablas interactivas y atractivas
  • dplyr: Facilita la manipulación de datos
  • DescTools: Contiene funciones estadísticas adicionales
  • ggplot2: Crea gráficos profesionales
2
Cargar y Explorar los Datos

Cargamos nuestros datos de estaturas y los exploramos:

# Cargar los datos (asumiendo que tienes un archivo CSV)
datos <- read.csv("Estatura_estudiantes.csv")

# O crear datos de ejemplo
set.seed(123)  # Para reproducibilidad
datos <- data.frame(
  Estaturas = round(rnorm(150, mean = 170, sd = 8), 1)

# Explorar los datos

# Crear vector con las estaturas

3
Calcular Estadísticas Básicas

Calculamos las estadísticas descriptivas fundamentales:

# Estadísticas básicas
N <- length(num)        # Cantidad de datos (tamaño muestral)
Dm <- max(num)          # Dato mayor (máximo)
Dn <- min(num)          # Dato menor (mínimo)
R <- (Dm - Dn)          # Rango

# Número de clases usando la regla de Sturges
K <- 1 + 3.322 * log10(N)  
I <- round(R / K)       # Intervalo de clase

# Mostrar resultados
cat("Estadísticas Básicas:\n")
cat("N (tamaño muestral):", N, "\n")
cat("Valor máximo:", Dm, "cm\n")
cat("Valor mínimo:", Dn, "cm\n")
cat("Rango:", R, "cm\n")
cat("Número de clases:", round(K), "\n")
cat("Intervalo de clase:", I, "cm\n")
Resultados
  • N: 150 estudiantes
  • Máximo: 185.2 cm
  • Mínimo: 152.1 cm
  • Rango: 33.1 cm
  • Clases: 8
  • Intervalo: 4 cm
¿Qué es la Regla de Sturges?
K = 1 + 3.322 × log₁₀(N)

Es una fórmula para determinar el número óptimo de intervalos (clases) en un histograma, basándose en el tamaño de la muestra.

4
Medidas de Tendencia Central

Calculamos la media, mediana y moda para datos agrupados:

# Crear intervalos y calcular frecuencias
lai <- seq(Dn, Dm, by = I)  # Límites aparentes inferiores
lai <- lai[lai <= Dm]
las <- lai + I - 1          # Límites aparentes superiores

# Límites reales
lri <- lai - 0.5
lrs <- las + 0.5

# Calcular frecuencias
frec <- Freq(num, breaks = c(lai, max(las) + 1), right = FALSE)
frecuencia <- frec$freq
frecuencia_acumulada <- cumsum(frecuencia)

# Puntos medios de los intervalos
xi <- (lai + las) / 2
fxi <- frecuencia * xi

# MEDIA ARITMÉTICA
media_agrupada <- sum(fxi) / N

# MEDIANA
n_2 <- N / 2
clase_mediana <- which(frecuencia_acumulada >= n_2)[1]
Li_med <- lri[clase_mediana]
Fi_anterior <- if(clase_mediana > 1) frecuencia_acumulada[clase_mediana - 1] else 0
fi_med <- frecuencia[clase_mediana]
A_med <- I

mediana_agrupada <- Li_med + ((n_2 - Fi_anterior) / fi_med) * A_med

# MODA
clase_modal <- which.max(frecuencia)
Li <- lri[clase_modal]
fi <- frecuencia[clase_modal]
fi_anterior <- if(clase_modal > 1) frecuencia[clase_modal - 1] else 0
fi_posterior <- if(clase_modal < length(frecuencia)) frecuencia[clase_modal + 1] else 0
A <- I

moda_agrupada <- Li + ((fi - fi_anterior) / ((fi - fi_anterior) + (fi - fi_posterior))) * A

# Mostrar resultados
cat("MEDIDAS DE TENDENCIA CENTRAL:\n")
cat("Media:", round(media_agrupada, 2), "cm\n")
cat("Mediana:", round(mediana_agrupada, 2), "cm\n")
cat("Moda:", round(moda_agrupada, 2), "cm\n")

5
Cuartiles y Medidas de Posición

Calculamos los cuartiles para entender cómo se distribuyen los datos:

# CUARTIL 1 (Q1)
n_4 <- N / 4
clase_Q1 <- which(frecuencia_acumulada >= n_4)[1]
Li_Q1 <- lri[clase_Q1]
fa_anterior_Q1 <- if(clase_Q1 > 1) frecuencia_acumulada[clase_Q1 - 1] else 0
f_Q1 <- frecuencia[clase_Q1]
A_Q1 <- I

Q1 <- Li_Q1 + ((n_4 - fa_anterior_Q1) / f_Q1) * A_Q1

# CUARTIL 3 (Q3)
n_3_4 <- 3 * (N / 4)
clase_Q3 <- which(frecuencia_acumulada >= n_3_4)[1]
Li_Q3 <- lri[clase_Q3]
fa_anterior_Q3 <- if(clase_Q3 > 1) frecuencia_acumulada[clase_Q3 - 1] else 0
f_Q3 <- frecuencia[clase_Q3]
A_Q3 <- I

Q3 <- Li_Q3 + ((n_3_4 - fa_anterior_Q3) / f_Q3) * A_Q3

# RANGOS
RQ <- Q3 - Q1      # Rango Intercuartílico
RQS <- RQ / 2      # Rango Semi-intercuartílico

cat("CUARTILES Y RANGOS:\n")
cat("Q1 (Cuartil 1):", round(Q1, 2), "cm\n")
cat("Q2 (Mediana):", round(mediana_agrupada, 2), "cm\n")
cat("Q3 (Cuartil 3):", round(Q3, 2), "cm\n")
cat("Rango Intercuartílico:", round(RQ, 2), "cm\n")
cat("Rango Semi-intercuartílico:", round(RQS, 2), "cm\n")
Cuartiles
  • Q1: 164.5 cm (25% de los datos están por debajo)
  • Q2: 169.8 cm (mediana, 50% por debajo)
  • Q3: 175.2 cm (75% de los datos están por debajo)
  • RIC: 10.7 cm
  • RSC: 5.35 cm
¿Qué nos dicen los cuartiles?

Los cuartiles dividen nuestros datos en 4 partes iguales. El rango intercuartílico (RIC) nos dice que el 50% central de los estudiantes tiene estaturas entre 164.5 cm y 175.2 cm.

6
Medidas de Dispersión

Calculamos medidas que nos indican qué tan dispersos están los datos:

# DESVIACIÓN MEDIA
xi_media <- xi - media_agrupada
abs_xi_media <- abs(xi_media)
f_abs_xi_media <- abs_xi_media * frecuencia
Des_M <- sum(f_abs_xi_media) / N

# VARIANZA Y DESVIACIÓN ESTÁNDAR
xi_media_cuadrado <- xi_media^2
f_xi_media_cuadrado <- frecuencia * xi_media_cuadrado
varianza <- sum(f_xi_media_cuadrado) / N
desviacion_estandar <- sqrt(varianza)

# COEFICIENTE DE VARIACIÓN
CV <- desviacion_estandar / media_agrupada

# ASIMETRÍA (usando el tercer momento)
xi_media_cubo <- xi_media^3
f_xi_media_cubo <- frecuencia * xi_media_cubo
asimetria <- sum(f_xi_media_cubo) / (N * desviacion_estandar^3)

# CURTOSIS (usando el cuarto momento)
xi_media_4 <- xi_media^4
f_xi_media_4 <- frecuencia * xi_media_4
curtosis <- sum(f_xi_media_4) / (N * desviacion_estandar^4)

cat("MEDIDAS DE DISPERSIÓN:\n")
cat("Desviación Media:", round(Des_M, 2), "cm\n")
cat("Varianza:", round(varianza, 2), "cm²\n")
cat("Desviación Estándar:", round(desviacion_estandar, 2), "cm\n")
cat("Coeficiente de Variación:", round(CV * 100, 2), "%\n")
cat("Asimetría:", round(asimetria, 3), "\n")
cat("Curtosis:", round(curtosis, 3), "\n")
Medidas de Dispersión
  • Desviación Media: 6.4 cm
  • Varianza: 64.2 cm²
  • Desviación Estándar: 8.0 cm
  • Coef. de Variación: 4.7%
  • Asimetría: 0.12
  • Curtosis: 2.89
Interpretación de las Medidas
  • CV = 4.7%: Baja variabilidad (< 15% es considerado bajo)
  • Asimetría ≈ 0: Distribución aproximadamente simétrica
  • Curtosis ≈ 3: Distribución similar a la normal (mesocúrtica)
7
Crear Visualizaciones

Creamos gráficos para visualizar nuestros datos:

# HISTOGRAMA
hist(num, 
     breaks = 10,
     main = "Histograma de Estaturas de Estudiantes",
     xlab = "Estatura (cm)",
     ylab = "Frecuencia",
     col = "skyblue",
     border = "darkblue")

# Agregar líneas de las medidas de tendencia central
abline(v = media_agrupada, col = "red", lwd = 2, lty = 1)
abline(v = mediana_agrupada, col = "green", lwd = 2, lty = 2)
abline(v = moda_agrupada, col = "blue", lwd = 2, lty = 3)

# Leyenda
legend("topright", 
       c("Media", "Mediana", "Moda"), 
       col = c("red", "green", "blue"), 
       lty = c(1, 2, 3), 
       lwd = 2)

# BOXPLOT (Diagrama de Cajas)
boxplot(num, 
        main = "Boxplot de Estaturas",
        ylab = "Estatura (cm)",
        col = "lightgreen",
        border = "darkgreen")

# Agregar líneas de cuartiles
abline(h = c(Q1, mediana_agrupada, Q3), col = "red", lty = 2)
text(x = 1.4, y = c(Q1, mediana_agrupada, Q3), 
     labels = c("Q1", "Q2", "Q3"), col = "blue")
Interpretación de los Gráficos
Histograma
  • Muestra la distribución de frecuencias
  • Forma aproximadamente normal
  • Mayor concentración alrededor de 170 cm
Boxplot
  • Visualiza los cuartiles y valores atípicos
  • La caja representa el 50% central de los datos
  • Los bigotes muestran la dispersión

Crear Tabla de Frecuencias Profesional

Organizamos todos nuestros resultados en una tabla interactiva:

# Crear dataframe con todos los resultados
tabla_frecuencias <- data.frame(
  "Intervalo" = paste(lai, "-", las),
  "Límites_Reales" = paste(round(lri, 1), "-", round(lrs, 1)),
  "Punto_Medio" = round(xi, 1),
  "Frecuencia" = frecuencia,
  "Frec_Acumulada" = frecuencia_acumulada,
  "Frec_Relativa" = round(frecuencia / N, 3),
  "Frec_Rel_Acum" = round(frecuencia_acumulada / N, 3)
)

# Crear tabla interactiva
datatable(tabla_frecuencias,
          options = list(
            pageLength = 10,
            autoWidth = TRUE,
            scrollX = TRUE
          ),
          rownames = FALSE,
          caption = "Tabla de Distribución de Frecuencias - Estaturas de Estudiantes")
