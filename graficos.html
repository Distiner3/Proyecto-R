El Poder de la Visualización de Datos

Los gráficos son fundamentales en el análisis de datos. Permiten identificar patrones, tendencias y anomalías que podrían pasar desapercibidas en tablas de números. En R, crear gráficos profesionales es sorprendentemente fácil.

"Una imagen vale más que mil palabras... y un buen gráfico vale más que mil números."
Tipos de Gráficos en R
Histogramas

Para mostrar la distribución de una variable continua

Gráficos de Dispersión

Para explorar la relación entre dos variables

Boxplots

Para visualizar la distribución y detectar valores atípicos

1
Gráficos de Dispersión (Scatter Plot)

Perfectos para explorar la relación entre dos variables numéricas.

Ejemplo Básico:
Código R
# Crear datos de ejemplo
X <- c(40, 50, 60, 70, 80, 90, 95)
Y <- c(25, 30, 40, 50, 60, 70, 85)

# Gráfico personalizado
plot(X, Y,
     main = "Comparación MetaScore vs User Review",
     xlab = "Meta Score",
     ylab = "User Review",
     col = "steelblue",
     pch = 19,
     cex = 1.5)
Parámetros de Personalización:
Parámetro Descripción Ejemplo
col Color de los puntos "red", "blue", "#FF5733"
pch Tipo de punto 19 (círculo), 1 (círculo vacío), 8 (asterisco)
cex Tamaño de los puntos 0.5 (pequeño), 1 (normal), 2 (grande)
frame Mostrar marco TRUE (sí), FALSE (no)
2
Histogramas

Muestran la distribución de frecuencias de una variable continua.

Código R
# Datos de ejemplo (estaturas de estudiantes)
estaturas <- c(165, 170, 168, 172, 175, 169, 171, 167, 173, 166, 174, 170, 168, 169, 172)

# Histograma personalizado
hist(estaturas,
     main = "Distribución de Estaturas de Estudiantes",
     xlab = "Estatura (cm)",
     ylab = "Frecuencia",
     col = "lightblue",
     border = "darkblue",
     breaks = 8)

# Agregar línea de la media
abline(v = mean(estaturas), col = "red", lwd = 2, lty = 2)
text(mean(estaturas) + 1.8, 3, "Media", col = "red")
Parámetros Importantes:
  • breaks: Número de barras (ej: breaks = 10)
  • col: Color de relleno (ej: "skyblue")
  • border: Color del borde (ej: "black")
  • freq: Mostrar frecuencias (TRUE) o densidad (FALSE)
  • xlim: Límites del eje X (ej: c(160, 180))
  • ylim: Límites del eje Y (ej: c(0, 5))
3
Boxplots (Diagramas de Caja)

Visualizan la distribución de datos mostrando cuartiles, mediana y valores atípicos.

Código R
# Crear datos de ejemplo
datos <- data.frame(
  sueldo = c(1200, 1500, 1800, 2000, 2200, 1600, 1900, 2100, 1400, 1700),
  grupo = c("A", "B", "A", "B", "A", "B", "A", "B", "A", "B")
)

# Boxplot agrupado por categorías
boxplot(sueldo ~ grupo,
        data = datos,
        main = "Distribución de Sueldos por Grupo",
        xlab = "Grupo",
        ylab = "Sueldo ($)",
        col = c("lightcoral", "lightblue"))
¿Cómo leer un Boxplot?
  • Línea central: Mediana (Q2)
  • Caja inferior: Primer cuartil (Q1)
  • Caja superior: Tercer cuartil (Q3)
  • Bigotes: Rango de datos normales
  • Puntos aislados: Valores atípicos
  • Altura de caja: Rango intercuartílico (IQR)

Personalización Avanzada

Técnicas para crear gráficos más profesionales y atractivos.

Añadir Múltiples Elementos:
Gráfico Completo
# Datos de ejemplo
x <- 1:10
y1 <- x * 2
y2 <- x * 1.5 + 1

# Crear gráfico base
plot(x, y1, type = "b", col = "blue", pch = 16, lwd = 2,
     main = "Comparación de Tendencias", xlab = "Tiempo",
     ylab = "Valores", ylim = c(0, 25))

# Agregar segunda línea
lines(x, y2, col = "red", type = "b", pch = 17, lwd = 2)

# Agregar leyenda
legend("topleft", legend = c("Serie 1", "Serie 2"),
       col = c("blue", "red"), pch = c(16, 17), lwd = 2)
