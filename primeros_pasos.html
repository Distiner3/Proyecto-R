¡Bienvenido a tus Primeros Pasos con R!

En esta sección aprenderás lo básico para comenzar a trabajar con R de manera efectiva. Conocerás el entorno de RStudio, aprenderás a manejar directorios, variables y realizarás tus primeras operaciones.

Prerrequisito: Asegúrate de haber completado la instalación de R y RStudio antes de continuar.
1
Conociendo la Interfaz de RStudio

RStudio tiene 4 paneles principales que debes conocer:

Editor de Scripts
Superior Izquierdo
Aquí escribes y guardas tu código R. Es como un bloc de notas inteligente para programadores.
Consola
Inferior Izquierdo
Donde se ejecuta el código y ves los resultados inmediatamente. También puedes escribir comandos directamente aquí.
Entorno
Superior Derecho
Muestra todas las variables, datos y objetos que has creado. Es como tu "inventario" de datos.
Archivos/Gráficos
Inferior Derecho
Explorador de archivos, visualizador de gráficos, ayuda y gestor de paquetes.
Consejo Pro

Puedes redimensionar los paneles arrastrando los bordes. Personaliza RStudio para que se adapte a tu forma de trabajar.

2
Manejo del Directorio de Trabajo

El directorio de trabajo es la carpeta donde R busca y guarda archivos por defecto.

Comandos Esenciales:
getwd()
[1] "C:/Users/TuUsuario/Documents"
Este comando muestra tu directorio actual
setwd("C:/Mi_Proyecto_R")
# Cambia al nuevo directorio
Cambiar a un directorio específico
list.files()
[1] "datos.csv" "mi_script.R" "resultados.txt"
Ver todos los archivos en el directorio actual
Importante

Siempre usa barras inclinadas "/" en las rutas, incluso en Windows. Ejemplo: "C:/Documentos/Proyecto" en lugar de "C:\Documentos\Proyecto"

3
Variables y Asignación de Valores

Las variables son contenedores que almacenan datos. En R puedes usar <- o = para asignar valores.

Tipos de Variables Básicas:
Números
edad <- 25
altura = 1.75
edad
altura
[1] 25
[1] 1.75
Texto
nombre <- "Ana"
ciudad = "Guatemala"
nombre
ciudad
[1] "Ana"
[1] "Guatemala"
Lógicos
es_estudiante <- TRUE
tiene_auto = FALSE
es_estudiante
[1] TRUE
Listas (Vectores)
numeros <- c(10, 20, 30)
colores = c("rojo", "azul")
numeros
[1] 10 20 30
Atajo de Teclado

Para escribir <- rápidamente, usa Alt + - (guión)

Buenas Prácticas para Nombres
  • Usa nombres descriptivos: peso_estudiante en lugar de x
  • No uses espacios: mi_variable no mi variable
  • No empieces con números: datos1 no 1datos
  • R distingue mayúsculas: Edad es diferente de edad

Trabajando con Listas (Vectores)

Los vectores son listas de elementos del mismo tipo. Son fundamentales en R:

Crear y Manipular Vectores
# Crear vectores
numeros <- c(10, 20, 30, 40, 50)
nombres <- c("Ana", "Luis", "María")

# Ver el contenido
numeros
nombres
[1] 10 20 30 40 50
[1] "Ana" "Luis" "María"
Funciones Útiles para Vectores
length(numeros) # Cantidad
max(numeros) # Mayor
min(numeros) # Menor
mean(numeros) # Promedio
sum(numeros) # Suma total
[1] 5
[1] 50
[1] 10
[1] 30
[1] 150
# Acceder a elementos
numeros[1] # Primer elemento
numeros[3] # Tercer elemento
numeros[c(1,3,5)] # Elementos 1, 3 y 5
[1] 10
[1] 30
[1] 10 30 50
¡Importante!

En R, los índices empiezan en 1, no en 0 como en otros lenguajes. El primer elemento es [1], no [0].

6
Crear tu Primer Script

Un script es un archivo que contiene código R. Te permite guardar y reutilizar tu trabajo:

Pasos para crear un script:
  1. En RStudio: File → New File → R Script (o Ctrl+Shift+N)
  2. Escribe tu código en la ventana que se abre
  3. Para ejecutar: selecciona el código y presiona Ctrl+Enter
  4. Para guardar: Ctrl+S y guarda con extensión .R
Ejemplo de Script Básico:
# Mi primer script en R
# Autor: [Tu nombre]
# Fecha: [Fecha actual]

# Datos de ejemplo
mi_nombre <- "Tu Nombre"
mi_edad <- 25
mis_hobbies <- c("Leer", "Deportes", "Música")

# Mostrar información
cat("Hola, soy", mi_nombre, "\n")
cat("Tengo", mi_edad, "años\n")
cat("Mis hobbies son:", paste(mis_hobbies, collapse = ", "), "\n")

# Cálculo simple
años_hasta_30 <- 30 - mi_edad
cat("Me faltan", años_hasta_30, "años para cumplir 30\n")
Usa Comentarios

Los comentarios (que empiezan con #) son fundamentales. Explican qué hace tu código y te ayudarán a recordar qué pensabas cuando lo escribiste.

Ejercicio Práctico: ¡Pon en Práctica lo Aprendido!
Crea un script que calcule tu IMC (Índice de Masa Corporal)

El IMC se calcula como: peso (kg) / altura (m)²

# Ejercicio: Calculadora de IMC

# Paso 1: Define tus datos
mi_peso <- ___ # Tu peso en kg
mi_altura <- ___ # Tu altura en metros (ej: 1.75)

# Paso 2: Calcula el IMC
mi_imc <- mi_peso / (mi_altura ^ 2)

# Paso 3: Muestra el resultado
cat("Mi IMC es:", round(mi_imc, 2), "\n")

# Paso 4: Interpreta el resultado
if (mi_imc < 18.5) {
  cat("Clasificación: Bajo peso\n")
} else if (mi_imc < 25) {
  cat("Clasificación: Peso normal\n")
} else {
  cat("Clasificación: Sobrepeso\n")
}
Solución del Ejercicio:
# Ejercicio: Calculadora de IMC - SOLUCIÓN

# Paso 1: Define tus datos
mi_peso <- 70 # Tu peso en kg
mi_altura <- 1.75 # Tu altura en metros

# Paso 2: Calcula el IMC
mi_imc <- mi_peso / (mi_altura ^ 2)

# Paso 3: Muestra el resultado
cat("Mi IMC es:", round(mi_imc, 2), "\n")

# Paso 4: Interpreta el resultado
if (mi_imc < 18.5) {
  cat("Clasificación: Bajo peso\n")
} else if (mi_imc < 25) {
  cat("Clasificación: Peso normal\n")
} else {
  cat("Clasificación: Sobrepeso\n")
}
Mi IMC es: 22.86
Clasificación: Peso normal
