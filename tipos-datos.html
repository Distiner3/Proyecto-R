¿Qué son los Tipos de Datos?

Los tipos de datos definen qué clase de información puede almacenar una variable. R automáticamente detecta el tipo, pero es importante entender cada uno para usar las funciones correctas y evitar errores.

¿Por qué es importante? Cada tipo de dato tiene funciones específicas. Por ejemplo, solo puedes calcular la media de números, no de texto.

1
Character (Texto)

Almacena texto, palabras, frases o cualquier cadena de caracteres. Siempre va entre comillas.

Ejemplos Básicos:
# Crear variables de texto
nombre <- "María"
apellido <- "González"
ciudad <- "Guatemala City"
mensaje <- "Hola, ¿cómo estás?"

# Ver el tipo
class(nombre)

# Ver el contenido
nombre
mensaje
[1] "character"
[1] "María"
[1] "Hola, ¿cómo estás?"
Funciones Útiles para Texto:
texto <- "Estadística con R"

nchar(texto) # Contar caracteres
toupper(texto) # Convertir a mayúsculas
tolower(texto) # Convertir a minúsculas
substr(texto, 1, 5) # Extraer caracteres (posición 1 a 5)

# Concatenar texto
paste(nombre, apellido) # Unir con espacio
paste(nombre, apellido, sep = "-") # Unir con guión
[1] 17
[1] "ESTADÍSTICA CON R"
[1] "estadística con r"
[1] "Estad"
[1] "María González"
[1] "María-González"
Funciones Importantes
  • nchar() - Contar caracteres
  • paste() - Unir textos
  • substr() - Extraer parte del texto
  • toupper() - A mayúsculas
  • tolower() - A minúsculas
  • gsub() - Reemplazar texto

Numeric (Números con Decimales)

Almacena números que pueden tener decimales. Es el tipo más común para cálculos matemáticos.

# Crear variables numéricas
altura <- 1.75
peso <- 70.5
temperatura <- 36.8
precio <- 15.99

# Ver el tipo
class(altura)

# Operaciones matemáticas
imc <- peso / (altura ^ 2)
imc

# Redondear
round(imc, 2)
[1] "numeric"
[1] 23.02041
[1] 23.02
Funciones Matemáticas Comunes:
numeros <- c(10.5, 15.2, 8.7, 12.1, 9.8)

mean(numeros) # Media
median(numeros) # Mediana
max(numeros) # Máximo
min(numeros) # Mínimo
sd(numeros) # Desviación estándar
sum(numeros) # Suma total
length(numeros) # Cantidad de elementos

[1] 10.5
[1] 15.2
[1] 8.7

Integer (Números Enteros)

Números sin decimales. Para crearlos explícitamente, usa L después del número o as.integer().

# Crear enteros explícitamente
edad <- 25L
estudiantes <- 150L

# Convertir a entero
numero_decimal <- 15.7
numero_entero <- as.integer(numero_decimal)

# Ver los tipos
class(edad)
class(numero_entero)

# Ver los valores
edad
numero_entero # Nota: se trunca, no se redondea
[1] "integer"
[1] "integer"
[1] 25
[1] 15
Importante: as.integer() trunca (corta) los decimales, no redondea. Para redondear usa round() primero.

Logical (Booleano)

Solo puede ser TRUE o FALSE. Resultado de comparaciones y operaciones lógicas.

# Crear variables lógicas
es_estudiante <- TRUE
tiene_beca <- FALSE

# Resultado de comparaciones
edad <- 20
es_mayor_edad <- edad>= 18

# Ver los tipos y valores
class(es_estudiante)
es_mayor_edad

# Operaciones lógicas
puede_votar <- es_mayor_edad & es_estudiante
puede_votar
[1] "logical"
[1] TRUE
[1] TRUE
Conversiones con Logical:
# Conversiones interesantes
as.numeric(TRUE) # TRUE = 1
as.numeric(FALSE) # FALSE = 0

as.logical(1) # 1 = TRUE
as.logical(0) # 0 = FALSE
as.logical(5) # Cualquier número ≠ 0 = TRUE

# Contar valores TRUE
valores <- c(TRUE, FALSE, TRUE, TRUE, FALSE)
sum(valores) # Cuenta los TRUE
[1] 1
[1] 0
[1] TRUE
[1] FALSE
[1] TRUE
[1] 3

Factor (Datos Categóricos)

Para variables categóricas como género, nivel educativo, etc. R los trata de manera especial en análisis estadísticos.

# Crear factores
genero <- factor(c("Masculino", "Femenino" , "Femenino" , "Masculino" ))
nivel_educativo <- factor(c("Básico", "Medio" , "Superior" , "Medio" ),
                               levels = c("Básico", "Medio", "Superior"))

# Ver el tipo y estructura
class(genero)
genero
levels(genero) # Ver las categorías

# Contar frecuencias
table(genero)
[1] "factor"
[1] Masculino Femenino Femenino Masculino
Levels: Femenino Masculino
[1] "Femenino" "Masculino"
genero
Femenino Masculino
        2          2
¿Cuándo usar Factor?
Usa Factor para:
  • Variables categóricas (género, color, región)
  • Variables ordinales (bajo, medio, alto)
  • Cuando quieras controlar el orden de las categorías
  • Para análisis estadísticos (ANOVA, regresión)

Funciones de Verificación y Conversión

Funciones para verificar tipos de datos y convertir entre ellos.

Funciones de Verificación:
# Crear variables de diferentes tipos
texto <- "Hola"
numero <- 15.5
entero <- 10L
logico <- TRUE

# Verificar tipos
is.character(texto) # ¿Es texto?
is.numeric(numero) # ¿Es numérico?
is.integer(entero) # ¿Es entero?
is.logical(logico) # ¿Es lógico?

# Función general
class(texto)
typeof(numero)
[1] TRUE
[1] TRUE
[1] TRUE
[1] TRUE
[1] "character"
[1] "double"
Funciones de Conversión:
# Conversiones comunes
numero_texto <- "123"

as.numeric(numero_texto) # Texto a número
as.character(456) # Número a texto
as.logical(1) # Número a lógico
as.integer(15.7) # Decimal a entero

# Conversión con vectores
numeros_texto <- c("10", "20" , "30" )
numeros_numericos <- as.numeric(numeros_texto)
numeros_numericos
class(numeros_numericos)
[1] 123
[1] "456"
[1] TRUE
[1] 15
[1] 10 20 30
[1] "numeric"
Funciones Clave
Verificar Convertir Descripción
is.character() as.character() Texto/cadenas de caracteres
is.numeric() as.numeric() Números con decimales
is.integer() as.integer() Números enteros
is.logical() as.logical() TRUE/FALSE
is.factor() as.factor() Variables categóricas
Ejercicio Práctico: Análisis de Estudiantes
Crea un pequeño dataset con información de estudiantes

Practica creando variables de diferentes tipos y analizándolas

# Ejercicio: Dataset de estudiantes

# Crea las siguientes variables:
nombres <- c("Ana", "Luis" , "María" , "Carlos" )
edades <- c(20, 22, 19, 21)
alturas <- c(1.65, 1.78, 1.60, 1.82)
es_becado <- c(TRUE, FALSE, TRUE, FALSE)
carrera <- factor(c("Medicina", "Ingeniería" , "Medicina" , "Derecho" ))

# Tu tarea:
# 1. Verifica el tipo de cada variable
# 2. Calcula la edad promedio
# 3. Encuentra la altura máxima
# 4. Cuenta cuántos estudiantes tienen beca
# 5. Crea una tabla de frecuencias de carreras
Solución del Ejercicio:
# Solución completa

# 1. Verificar tipos
class(nombres) # "character"
class(edades) # "numeric"
class(alturas) # "numeric"
class(es_becado) # "logical"
class(carrera) # "factor"

# 2. Edad promedio
edad_promedio <- mean(edades)
cat("Edad promedio:", edad_promedio, "años\n")

# 3. Altura máxima
altura_maxima <- max(alturas)
cat("Altura máxima:", altura_maxima, "m\n")

# 4. Contar becados
total_becados <- sum(es_becado)
cat("Estudiantes con beca:", total_becados, "\n")

# 5. Tabla de carreras
tabla_carreras <- table(carrera)
print(tabla_carreras)
