¿Qué vamos a instalar?

Para comenzar a trabajar con R necesitamos instalar dos programas esenciales:

R

El lenguaje de programación y entorno estadístico principal.

RStudio

Un entorno de desarrollo integrado (IDE) que hace más fácil trabajar con R.

Requisitos del Sistema
Windows
  • Windows 10 o superior
  • 2 GB RAM mínimo
  • 500 MB espacio libre
macOS
  • macOS 10.15 o superior
  • 2 GB RAM mínimo
  • 500 MB espacio libre
Linux
  • Ubuntu 18.04+ / CentOS 7+
  • 2 GB RAM mínimo
  • 500 MB espacio libre
1
Descargar e Instalar R

R es el núcleo del sistema. Siempre debemos instalarlo primero.

Descargar R para Windows
Importante

Descarga siempre la versión más reciente desde el sitio oficial de CRAN. Evita descargas de sitios no oficiales.

Pasos de Instalación:
  1. Haz clic en el enlace de descarga
  2. Selecciona "Download R for Windows"
  3. Haz clic en "base"
  4. Descarga el archivo .exe
  5. Ejecuta el instalador como administrador
  6. Acepta todas las opciones predeterminadas
  7. Espera a que termine la instalación
2
Descargar e Instalar RStudio

RStudio es nuestro entorno de trabajo visual. Hace que trabajar con R sea mucho más fácil.

Descargar RStudio Desktop
Pasos de Instalación:
  1. Haz clic en el enlace de descarga
  2. Selecciona "RStudio Desktop" (versión gratuita)
  3. Elige tu sistema operativo
  4. Descarga el archivo instalador
  5. Ejecuta el instalador
  6. Sigue las instrucciones en pantalla
  7. Deja las opciones predeterminadas
3
Primera Configuración

Ahora vamos a abrir RStudio y hacer algunas configuraciones básicas.

Abriendo RStudio por primera vez:
  1. Busca "RStudio" en el menú de inicio
  2. Haz clic para abrir la aplicación
  3. RStudio debería detectar automáticamente R
  4. Verás una interfaz dividida en varias ventanas
Verificar que todo funciona:

Escribe este comando en la Consola (ventana inferior izquierda):

print("¡Hola, R!")

Si aparece [1] "¡Hola, R!" ¡felicidades! Todo está instalado correctamente.

4
Personalizar RStudio

Vamos a personalizar RStudio para que sea más cómodo trabajar.

Cambiar el tema y apariencia:
  1. Ve a Tools → Global Options
  2. En la sección Appearance:
    • Editor theme: Elige un tema que te guste (ej: "Cobalt", "Tomorrow Night")
    • Editor font size: Cambia el tamaño (recomendado: 12-14)
    • Editor font: Cambia la fuente si deseas
  3. Haz clic en Apply y OK
Consejo

Un tema oscuro puede ser más cómodo para sesiones largas de trabajo. Los temas claros son mejores para presentaciones.

5
Configurar Directorio de Trabajo

El directorio de trabajo es la carpeta donde R buscará y guardará archivos por defecto.

Ver directorio actual:
getwd()
Cambiar directorio:
setwd("C:/Mi_Proyecto_R")
Ver archivos en el directorio:
list.files()
Recomendación

Crea una carpeta específica para tus proyectos de R, por ejemplo "Documentos/Proyectos_R". Usa siempre barras inclinadas "/" en las rutas, incluso en Windows.

¡Instalación Completada!

Excelente, ya tienes R and RStudio instalados y configurados. Ahora estás listo para comenzar a programar.
