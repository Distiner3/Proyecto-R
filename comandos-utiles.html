Referencia Rápida de Comandos en R

Esta es tu cheat sheet completa de R. Encuentra rápidamente la función que necesitas, con ejemplos prácticos y explicaciones claras. Usa el buscador para encontrar comandos específicos.

Consejo: Guarda esta página en tus favoritos. Te servirá como referencia rápida mientras programas en R.
Funciones Matemáticas
Función Descripción Ejemplo Resultado
log() Logaritmo natural log(10) 2.302585
abs() Valor absoluto abs(-5) 5
sin(), cos() Funciones trigonométricas sin(pi/2) 1
round() Redondear valores round(3.14159, 2) 3.14
sqrt() Raíz cuadrada sqrt(16) 4
ceiling() Redondeo hacia arriba ceiling(3.2) 4
floor() Redondeo hacia abajo floor(3.8) 3
trunc() Eliminar parte decimal trunc(3.14159) 3
Tip: Usa round(x, digits = n) para controlar el número de decimales.
Manipulación de Texto
Función Descripción Ejemplo Resultado
nchar() Contar caracteres nchar("Hola") 4
tolower() Convertir a minúsculas tolower("HOLA") "hola"
toupper() Convertir a mayúsculas toupper("hola") "HOLA"
substr() Extraer texto substr("manzana", 1, 4) "manz"
sub() Sustituir primer valor sub("a", "A", "casa") "cAsa"
gsub() Sustituir todos los valores gsub("a", "A", "casa") "cAsA"
paste() Concatenar strings paste("Hola", "mundo") "Hola mundo"
strsplit() Separar texto strsplit("a,b,c", ",") ["a","b","c"]
Tip: Usa paste(..., sep = "") para unir sin espacios, o paste0() como atajo.
Trabajo con Vectores
Función Descripción Ejemplo Resultado
c() Crear vector c(1, 2, 3, 4) [1 2 3 4]
length() Cantidad de elementos length(c(1,2,3)) 3
max() Valor máximo max(c(1,5,3)) 5
min() Valor mínimo min(c(1,5,3)) 1
mean() Promedio mean(c(1,2,3)) 2
sum() Suma todos los valores sum(c(1,2,3)) 6
sort() Ordenar de menor a mayor sort(c(3,1,2)) [1 2 3]
which() Posiciones donde condición es TRUE which(c(1,5,3) > 2) [2 3]
union() Unir vectores sin repetir union(c(1,2), c(2,3)) [1 2 3]
intersect() Elementos en común intersect(c(1,2), c(2,3)) [2]
setdiff() Elementos que tiene A y no B setdiff(c(1,2), c(2,3)) [1]
%in% Verificar si está en el vector "a" %in% c("a","b") TRUE
Tip: Usa sort(x, decreasing = TRUE) para ordenar de mayor a menor.
Trabajo con DataFrames
Función Descripción Ejemplo Resultado
data.frame() Crear DataFrame data.frame(x=c(1,2), y=c(3,4)) Tabla con columnas x, y
View() Ver datos en ventana View(datos) Abre ventana con tabla
head() Primeras 6 filas head(datos) Muestra primeras filas
nrow() Número de filas nrow(datos) Cantidad de filas
ncol() Número de columnas ncol(datos) Cantidad de columnas
str() Estructura de la tabla str(datos) Tipos de datos y preview
summary() Resumen estadístico summary(datos) Min, Max, Mean, etc.
rbind() Agregar filas rbind(df1, df2) Combina por filas
table() Tabla de frecuencias table(datos$columna) Cuenta valores únicos
Tip: Usa datos$columna para acceder a una columna específica.
Manejo de Archivos
Función Descripción Ejemplo
getwd() Ver directorio actual getwd()
setwd() Cambiar directorio setwd("C:/Mi_Proyecto")
list.files() Listar archivos del directorio list.files()
read.csv() Leer archivo CSV read.csv("datos.csv")
write.csv() Escribir archivo CSV write.csv(datos, "salida.csv")
write.table() Escribir tabla con opciones write.table(datos, "archivo.csv", sep=";")
scan() Leer datos simples scan("numeros.txt")
Tip: Usa row.names = FALSE en write.csv para evitar números de fila.
Scripts y Control de Flujo
Función Descripción Ejemplo
rm() Eliminar objetos rm(variable)
ls() Listar objetos en memoria ls()
cat() Imprimir texto cat("Hola mundo\n")
print() Imprimir objetos print(datos)
help() Obtener ayuda help(mean) o ?mean
install.packages() Instalar paquetes install.packages("ggplot2")
library() Cargar paquetes library(ggplot2)
Tip: Usa Ctrl+Enter para ejecutar la línea actual en un script.
Estadísticas y Análisis
Función Descripción Ejemplo Uso
mean() Media aritmética mean(datos$edad) Tendencia central
median() Mediana median(datos$edad) Tendencia central
sd() Desviación estándar sd(ventas) Dispersión
var() Varianza var(ventas) Dispersión
quantile() Cuartiles y percentiles quantile(datos, 0.25) Posición
cor() Correlación de Pearson cor(x, y) Relación lineal
cov() Covarianza cov(a, b) Variación conjunta
hist() Histograma hist(datos$variable) Distribución
boxplot() Diagrama de cajas boxplot(y ~ x, data=df) Comparar grupos
plot() Gráfico de dispersión plot(x, y) Relación entre variables
Tip: Usa na.rm = TRUE en funciones estadísticas para ignorar valores faltantes.
Conversiones de Tipos
Función Descripción Ejemplo
as.character() Convertir a texto as.character(5.6) → "5.6"
as.numeric() Convertir a número as.numeric("520") → 520
as.integer() Convertir a entero as.integer(5.7) → 5
as.logical() Convertir a lógico as.logical(10) → TRUE
as.factor() Convertir a factor as.factor(c("A","B","A"))
Tip: Siempre verifica el tipo con class() antes y después de convertir.
